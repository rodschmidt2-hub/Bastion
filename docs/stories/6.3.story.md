# Story 6.3 — Dashboard NPS Global e Alertas

**Epic:** 6 — Dashboard e Métricas
**Status:** Done
**PRD refs:** FR-67, FR-68

## Descrição
Como admin, quero ver no dashboard o NPS médio da base e uma lista de clientes com NPS baixo em alerta, para agir proativamente antes de perder clientes insatisfeitos.

## Acceptance Criteria

- [ ] Card "NPS Médio Global": média das últimas notas por cliente (uma nota por cliente, a mais recente) (FR-67)
- [ ] Semáforo NPS global: verde (≥ 8), âmbar (6–7), vermelho (< 6)
- [ ] Lista "Clientes em Alerta NPS": clientes com última nota ≤ 6, ordenados por score crescente (FR-68)
- [ ] Cada item da lista exibe: nome do cliente, score, data do registro, responsável
- [ ] Lista clicável — leva para o perfil do cliente na tab NPS
- [ ] Clientes sem nenhum registro NPS não entram no cálculo nem na lista de alertas

## Tarefas Técnicas

- [ ] Query: NPS médio global — `DISTINCT ON (cliente_id)` última nota por cliente, AVG do resultado
- [ ] Query: clientes em alerta — última nota por cliente WHERE score <= 6, ORDER BY score ASC
- [ ] Componente `nps-global-card.tsx`: card com score médio + semáforo
- [ ] Componente `nps-alertas-list.tsx`: lista de clientes com NPS baixo
- [ ] Adicionar seção NPS no `dashboard/page.tsx`

## Notas Técnicas
- Sem migration: usa `nps_registros` (Story 10.1)
- NPS médio usa apenas a última nota de cada cliente — não média de todos os registros históricos
- `DISTINCT ON (cliente_id) ORDER BY cliente_id, data_registro DESC` para última nota por cliente
- Semáforo global segue faixas NPS padrão: Promotores ≥ 9, Neutros 7–8, Detratores ≤ 6

## Definição de Pronto
- Card NPS médio global exibe valor e semáforo corretos
- Lista de alertas mostra apenas clientes com score ≤ 6
- Clique no cliente redireciona para tab NPS do perfil
