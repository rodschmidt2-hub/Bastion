# Story 5.4 — Exportação CSV do Extrato Financeiro

**Epic:** 5 — Financeiro e Cobranças
**Status:** Done
**PRD refs:** FR-36

## Descrição
Como admin, quero exportar o extrato financeiro de um cliente em CSV com todas as faturas e pagamentos, para conciliação contábil e relatórios externos.

## Acceptance Criteria

- [ ] Botão "Exportar CSV" na tab Financeiro do cliente
- [ ] CSV exportado contém: competência, número fatura, data emissão, data vencimento, valor total, valor pago, saldo devedor, status, pagamentos (data + valor + forma)
- [ ] Filtro de período antes do export: data início e data fim (por competência)
- [ ] Nome do arquivo: `extrato-{nome-cliente}-{periodo}.csv`
- [ ] Export com dados reais em tempo real (sem cache)
- [ ] Exportação na página `/financeiro` global: CSV de todos os clientes com saldo em aberto

## Tarefas Técnicas

- [ ] Route Handler: `GET /api/export/financeiro/[clienteId]` — gera e retorna CSV
- [ ] Query: faturas + fatura_itens + pagamentos do cliente no período
- [ ] Componente `export-csv-button.tsx`: botão com date range picker + trigger download
- [ ] Route Handler: `GET /api/export/financeiro/inadimplentes` — CSV global de inadimplência
- [ ] Adicionar botão de export na tab Financeiro e na página `/financeiro` global

## Notas Técnicas
- Sem migration
- Usar `papaparse` ou construção manual de CSV (evitar dependência pesada para caso simples)
- Route Handler retorna `Content-Type: text/csv` + `Content-Disposition: attachment`
- Encoding UTF-8 com BOM para compatibilidade com Excel brasileiro
- Valores monetários no formato `1234,56` (vírgula decimal) para Excel pt-BR

## Definição de Pronto
- CSV baixa corretamente com dados reais do cliente
- Filtro de período funciona
- Encoding correto (abre sem caracteres estranhos no Excel)
